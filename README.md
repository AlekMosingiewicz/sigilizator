SIGILIZATOR!!!
-------------------
-------------------

What is it?
-------------------

This purposefully awkward name hides an attempt to put technology in service of sorcery. As we know, Austin Osman Spare suggested that desires can be manifested by writing down what we desire, reducing it to incomprehensible phrase without recurring letters, and then creating a graphical representation out of it, e.g.
"I want lots of money"
becomes
"iwantlosfmey"
Those letters are then combined to create a sigil. What's the bottleneck? Getting rid of recurring letters. This program does just that.

How to use it?
--------------------

After building and running the program you will be prompted for the phrase to be sigilized, enter it (as in "I want lots of money"). The program will then output the set of letters that can be used to be combined into a sigil.
